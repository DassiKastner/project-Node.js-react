# חנות משחקי ילדים

**חנות משחקי ילדים** הוא אתר למכירת משחקים לילדים ותינוקות, עם דגש על נראות מודרנית וחוויית משתמש נעימה.

                                                                                                                                                                                                                  ---

## תיאור הפרויקט
האתר כולל את העמודים והתכונות הבאים:

- **דף הבית** – הצגת המוצרים של החנות עם אפשרות להוסיף אותם לסל.  
  *למשתמשים לא רשומים אין אפשרות להוסיף לסל.*
- **סל קניות** – תצוגה של המוצרים שנוספו לסל.  
- **ניהול מוצרים** – אפשרות להוסיף, לערוך ולמחוק מוצרים.  
- **הרשמה וכניסה** – יצירת משתמשים וניהול טוקן שמאוחסן ב־localStorage.
- **שמירת נתונים** – כל המידע של המוצרים והמשתמשים נשמר ב־MongoDB.

הדגש בפרויקט הוא על **עיצוב נקי ומודרני** באמצעות Material UI וחוויית משתמש נוחה ואינטואיטיבית.

                                                                                                                                                                                                                  ---

## טכנולוגיות
-**קליינט:** React, Material UI
-**שרת:** Node.js
-**מסד נתונים:** MongoDB

                                                                                                                                                                                                                  ---

## תכונות עיקריות
- הצגת מוצרים בדף הבית  
- הוספה והסרה של מוצרים מהסל  
- ניהול מלא של מוצרים (CRUD)  
- הרשמה וכניסה למשתמשים  
- שמירה של טוקן משתמש ב־localStorage  
- כל הנתונים נשמרים ב־MongoDB  
- עיצוב נקי ומודרני עם Material UI
